# Gerenciador de Playlist em Java

Este projeto permite criar e gerenciar uma playlist de músicas.
Funcionalidades:
- Adicionar músicas (título, artista, duração)
- Remover músicas pelo título
- Listar todas as músicas
- Mostrar duração total da playlist

Tecnologias usadas:
- Java
- ArrayList para armazenamento das músicas
- Scanner para entrada de dados no console

Arquivos:
- Musica.java -> Classe representando uma música
- Playlist.java -> Classe que gerencia a playlist
- Main.java -> Classe principal com a interface do usuário
